# A股盯盘软件

一个简洁的 Windows 悬浮窗 A 股实时行情盯盘工具，支持分时图、成交量、MACD、委比、五档行情及多周期顶底背离检测。

## 功能特点

- **悬浮窗显示**：无边框置顶小窗口，卡片式股票列表
- **实时行情**：自动刷新价格、涨跌幅、委比（非交易时间自动降频）
- **分时图**：鼠标悬浮弹出当日分时走势图
- **成交量 + MACD**：分时图内嵌成交量柱状图和1分钟MACD
- **五档行情**：分时图右侧显示买一到买五、卖一到卖五及委比
- **顶底背离检测**：支持 5分/15分/30分 周期的顶背离、底背离自动检测与提醒
- **请求限流**：内置请求频率限制，避免IP被封
- **智能定位**：分时图自动避让屏幕边界
- **自由拖动**：按住窗口任意位置可拖动
- **可配置**：设置窗口自定义股票、界面、指标显示等
- **零依赖**：仅使用 Python 标准库，无需安装额外包

## 使用方法

### 1. 启动程序

```bash
pythonw stock_monitor.py
```

或双击 `run_hidden.vbs` 启动。

### 2. 交互操作

| 操作 | 说明 |
|------|------|
| 按住拖动 | 拖动悬浮窗位置 |
| 鼠标悬浮股票 | 弹出分时走势图（含成交量+MACD+五档+背离提醒） |
| 右键菜单 | 设置 / 立即刷新 / 退出 |

### 3. 设置窗口（右键 → 设置）

**全局设置**：
- **透明度**：0.30 ~ 1.00 滑动条调节
- **字号**：小 / 中 / 大
- **刷新间隔**：1~60秒（非交易时间自动延长至30秒）
- **显示委比**：主窗口是否显示委比
- **显示五档行情**：分时图是否显示五档行情面板

**自选股设置**：
- 添加/删除股票（代码可自动识别 sh/sz）
- 每只股票可独立设置：是否启用顶底背离检测、周期（5分/15分/30分）

**股票数量建议**：不超过 10 只。

## 界面说明

- **红色**：上涨 / 委比为正 / MACD红柱
- **绿色**：下跌 / 委比为负 / MACD绿柱
- **蓝色高亮**：股票代码

分时图中：
- **蓝色线**：实时价格
- **橙色线**：均价
- **虚线**：昨收价
- **白色线**：MACD的DIF
- **黄色线**：MACD的DEA

标题栏提示：
- `⚡15分底`：15分钟周期底背离
- `⚠5分顶`：5分钟周期顶背离

## 配置示例

`config.json`：
```json
{
    "stocks": [
        {
            "code": "600519",
            "market": "sh",
            "name": "贵州茅台",
            "divergence": {
                "enabled": true,
                "periods": [5, 15, 30],
                "types": ["bottom", "top"]
            }
        }
    ],
    "window": {
        "x": 100,
        "y": 100,
        "refresh_interval": 3000
    },
    "ui": {
        "scale": 1.0,
        "alpha": 0.95,
        "show_weibi": true,
        "show_wudang": true
    }
}
```

## 注意事项

- 程序仅在工作日开盘期间有实时数据，非交易时间价格可能不变
- 数据来源于腾讯/东方财富公开接口，仅供个人参考
- 顶底背离检测基于历史多周期K线计算，存在滞后性，不构成投资建议
- 如需开机启动，可将 `run_hidden.vbs` 放入 Windows 启动文件夹
